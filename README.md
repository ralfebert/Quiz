# Quiz

iOS-Beispielprojekt welches ein Multiple-Choice-Quiz realisiert.

Varianten:

* [`starter`](https://github.com/ralfebert/Quiz/tree/starter): Storyboard + UIViewController-Grundgerüst
* [`simple`](https://github.com/ralfebert/Quiz/tree/simple): Einfachstmögliche UIKit-Implementierung
* [`master`](https://github.com/ralfebert/Quiz/tree/master): Zustandsbasierte Variante mit parameterisiertem Enum `QuizState`
